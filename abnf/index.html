ABNF to REGEX Regular Expression Generator

Download ABNF to REGEX Github Project Issue Tracker Online Form

The ABNF to REGEX library is a pair of PHP functions that validate standard ABNF (RFC 5234) input and generate POSIX and Perl-compatible regular expressions for use in schema and applications. The online form uses it to provide a simple web-based front-end to these functions.
