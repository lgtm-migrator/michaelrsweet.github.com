RasterView CUPS/PWG/Apple Raster File Viewer

Download RasterView Github Project Issue Tracker

RasterView is a CUPS, PWG, and Apple raster file viewer. It basically allows you to look at the raster data produced by any of the standard CUPS RIP filters (cgpdftoraster, imagetoraster, pdftoraster, and pstoraster) and is normally used to either test those filters or look at the data that is being sent to your raster printer driver.

RasterView is provided as free software under version 2 of the GNU General Public license.
